🛡️ H4CKTHOR HUB - Monster Edition
H4CKTHOR HUB, Python və CustomTkinter kitabxanası ilə hazırlanmış, müasir interfeysə malik çoxfunksiyalı Sızma Testi (Penetration Testing) platformasıdır. Bu alət həm pasiv kəşfiyyat (recon), həm də aktiv hücum simulyasiyalarını tək bir mərkəzdən idarə etmək üçün nəzərdə tutulmuşdur.

⚠️ XƏBƏRDARLIQ: Bu alət yalnız təhsil və etik sızma testi məqsədləri üçün nəzərdə tutulub. İcazəsiz sistemlərdə istifadə edilməsi qanunsuz ola bilər. İstifadəçi bütün hərəkətlərinə görə özü məsuliyyət daşıyır.

🚀 Özəlliklər
H4ckthor Hub 11 fərqli kritik təhlükəsizlik modulu ilə təchiz olunub:

🔍 Pasiv Kəşfiyyat (Passive Recon)
Super Fast Port Scan: Hədəf IP/Host üzərində açıq portları tapır və servis bannerlərini (SSH, HTTP və s.) oxuyur.

Reverse IP Lookup: Eyni IP ünvanında yerləşən digər saytları müəyyən edir.

Subdomain Finder: Hədəf domenə bağlı alt domenləri (subdomains) siyahılayır.

WAF Detector: Saytın arxasında Cloudflare, Sucuri və ya ModSecurity kimi təhlükəsizlik divarlarının olub-olmadığını yoxlayır.

JS Secret Leaks: JavaScript fayllarını skan edərək daxili API açarlarını və gizli şifrələri axtarır.

⚔️ Aktiv Hücum (Active Attack)
Auto-Crawler Pro: Saytı avtomatik gəzərək bütün linkləri və URL parametrlərini tapır.

Mass Shell Hunter: Sunucuda kritik fayl yükləmə (upload) boşluqlarını axtarır.

SQLi Auto-Tester: SQL injection zəifliyini aşkar etmək üçün avtomatik payload-lar göndərir.

Admin Panel Finder: Geniş siyahıdan istifadə edərək saytın idarəetmə panelini (admin page) tapmağa çalışır.

🛠️ Sistem Alətləri (Utils)
Exploit Architect (AI): Daxili bazasından istifadə edərək versiya nömrələrinə uyğun CVE kodlarını və istismar üsullarını təklif edir.

Generate Report: Skan nəticələrini zaman damğası ilə .txt formatında yadda saxlayır.
